# Scientific Calculator (JavaScript)

A simple scientific calculator built using HTML, CSS, and JavaScript.

## 🚀 Features
- Basic arithmetic operations (+, -, ×, ÷)
- Scientific functions (sin, cos, tan, log, etc.)
- Clean UI design
- Responsive layout

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Run
Just open `index.html` in browser.
